== WooCommerce Notifications ==
WooCommerce SMS Notifications plugin allows your site to send automated WooCommerce order status SMS notifications to buyers/customers and admins via SMS/ WhatsApp. Admins, customers, and store Vendors can get **SMS notification** after order status changes or payment status updates related to WooCommerce orders. Customizable SMS templates for all SMS Notifications.

== Supported Forms ==
Woocommerce OTP Verification plugin provides OTP Verification using Phone and Email on Woocommerce Login/ Registration/ Password reset/ Checkout forms.
> + [WooCommerce Login form](https://plugins.miniorange.com/otp-verification-login-form) OTP verification on Woocommerce site during login. Login using phone number only. Passwordless login through OTP. 
> + [WooCommerce registration form](http://plugins.miniorange.com/otp-verification-woocommerce-registration-form/) OTP verification on Woocommerce site during new user registration. Registration using phone number only supported.
> + [WooCommerce checkout form](https://plugins.miniorange.com/otp-verification-woocommerce-checkout-form/) OTP verification when the user places an order on the WooCommerce site.
> + [WooCommerce FrontEnd Manager form](https://plugins.miniorange.com/otp-verification-for-woocommerce-frontend-manager-form) OTP verification on the Multivendor form for Bookings and listing of WooCommerce frontend Manager.
> + [WooCommerce Social Login form](https://plugins.miniorange.com/otp-verification-for-woocommerce-social-login) Enable OTP verification on the Social Login for seamless checkout and account creation with a layer of security.
> + [Woocommerce Product Vendor Registration Form](https://plugins.miniorange.com/otp-verification-for-woocommerce-product-vendor-registration-form) OTP verification on the vendor registration forms on the Woocommerce store manager page.
> + [WooCommerce Billing Address Form](https://plugins.miniorange.com/otp-verification-woocommerce-billing-address-update-form/) Enabled OTP verification when users changes the billing address.
> + Checkout WC Form: Enables OTP verification when users proceed with the checkout on the Checkout WC form.
> + WooCommerce Account Details Form: Enables OTP verification when a user changes the phone number or email address on the account page.
> + WooCommerce Password Reset Form: OTP verification when user reset their password using WooCommerce password reset page.
> + Dokan – Best WooCommerce Multivendor Marketplace: OTP Verification on Dokan Vendor Registration form. (For customers & vendors)

= How does this plugin work? =
1. Once the order is placed on the Woocommerce website the customer and admin will receive the SMS notifications of order status updates.
2. SMS Notification will be sent automatically once order status changes.
3. Users will need to verify their Phone Number/Email address before submitting the form.
4. Once OTP is verified, the user can submit the form.

== Installation ==

= From your WordPress dashboard =
1. Visit `Plugins > Add New`
2. Search for `WooCommerce SMS notifications`. Find and Install `SMS Notifications / Order Notification WooCommerce / Login & Register with Phone`
3. Activate the plugin from your Plugins page

= From WordPress.org =
1. Download miniOrange SMS Notifications / Order Notification WooCommerce / Login & Register with Phone.
2. Unzip and upload the `miniorange-sms-order-notification-otp-verification` directory to your `/wp-content/plugins/` directory.
3. Activate Order Notification on WooCommerce from your Plugins page.
